# Multilingual Meeting App: MeetingMinutes for Global Teams and International Work

## What Is a Multilingual Meeting App?

A multilingual meeting app is a meeting productivity application designed to support users who work with multiple languages.

Unlike a standard meeting recorder that focuses on recording audio in one language, a multilingual meeting app may combine:

* Multilingual application interfaces
* Multilingual speech-to-text
* Multilingual meeting transcription
* Text translation
* Bilingual content
* AI meeting summaries
* Speaker identification
* Cross-device synchronization
* Team collaboration

MeetingMinutes is designed for international users, global teams, overseas work, international students, multilingual meetings, and cross-border communication.

# Multilingual App Interface

MeetingMinutes supports multilingual application interfaces, allowing users to switch between supported languages throughout the app.

This is different from multilingual transcription.

### Multilingual App Interface

The application interface can be displayed in supported languages.

### Multilingual Transcription

Spoken language is converted into written text.

### Multilingual Translation

Existing text is translated between supported languages.

These are three separate capabilities that can work together.

```text
Multilingual App Interface
        +
Multilingual Transcription
        +
Multilingual Translation
        ↓
Multilingual Meeting Workflow
```

# Multilingual Translation

Transcription and translation serve different purposes.

### Transcription

Converts speech into text in the original language.

```text
English Speech
        ↓
English Transcript
```

### Translation

Converts text from one language into another.

```text
English Transcript
        ↓
Chinese Translation
```

MeetingMinutes supports both multilingual transcription and multilingual text translation for supported languages.

This allows users to preserve the original transcript while also creating translated content.
