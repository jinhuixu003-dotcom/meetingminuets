# Audio and Video Import: Import Existing Recordings for Transcription with MeetingMinutes

## What Is Audio and Video Import?

Audio and video import allows users to bring existing media files into MeetingMinutes for processing.

Not every conversation is recorded directly inside a meeting transcription app.

Users may already have recordings stored on:

* Smartphones
* Computers
* Voice recorders
* Cameras
* Messaging apps
* Video platforms
* Cloud storage
* Other recording applications

MeetingMinutes supports multiple audio and video import methods, allowing users to bring existing recordings into the app without relying on separate third-party format conversion tools for supported formats.

After importing a supported file, users can use available MeetingMinutes features such as:

* Speech-to-text transcription
* AI processing
* Meeting summaries
* Meeting minutes
* Speaker identification
* Translation
* Keyword search
* File management

The exact features available after importing a file may depend on the file format and application version.

---

# Why Import Audio and Video Files?

Users often have recordings that were created before they started using MeetingMinutes.

# Why Use Audio and Video Import in MeetingMinutes?

Many users already have recordings stored in different places and formats.

MeetingMinutes helps bring supported existing recordings into a broader workflow:

**Import → Transcribe → Identify Speakers → Search → Summarize → Create Meeting Minutes → Share**

With support for multiple audio and video formats, MeetingMinutes can help users process existing recordings without requiring separate format conversion tools for supported files.

Combined with transcription, multilingual processing, AI summaries, speaker identification, meeting minutes, file management, and cross-device synchronization, MeetingMinutes provides a workflow for turning existing audio and video recordings into structured, searchable information.
